This directory is for your application assets (logos, screenshots, installation guides).

Structure:
assets/
  images/
    logos/
      app-logo.svg
    overview/
      screenshot-1.jpg
    installation/
      step-1.jpg

NOTE: The file names and paths must match exactly what you reference in your metadata.json.
Example: "src": "/images/logos/app-logo.svg" corresponds to assets/images/logos/app-logo.svg.
